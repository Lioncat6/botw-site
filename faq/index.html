Frequently Asked Questions

How do I import the MCWorld File?

Here are the steps to import the MCWorld file for different platforms:

  • PC: Download the .mcworld file and double-click it to open it with Minecraft. The game will automatically import the world.
  • Android: Download the .mcworld file and open it using a file manager. Select Minecraft to open the file, and the game will import the world.
  • iOS: Download the .mcworld file and use the "Open in" option to select Minecraft. The game will import the world.
  • Console: There are no plans at the moment to list the world on the Minecraft marketplace. The only way to play this map on console is either through a realm or having another compatible device host the world.
